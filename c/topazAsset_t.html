(Class)

topazAsset_t 



    An asset represents a useful abstraction for an object 
    that was populated from raw data, usually from a file or 
    from memory.

    On it's own, it does not do much. But, it serves as a
    base class for more useful applications

(Enumerator)

topazAsset_Type 

 Type of asset.

 Every asset has a type that gives a hint to what child class
 it is.
topazAsset_Type_None The asset doesnt have a type. This is used for error cases.
topazAsset_Type_Image The asset is an image. See image.h
topazAsset_Type_Sound The asset is an audio waveform.
topazAsset_Type_Material The asset defines how a 3D mesh should be expressed.
topazAsset_Type_Particle The asset is a particle specification.
topazAsset_Type_Data The asset is raw bytes / text.
topazAsset_Type_Mesh The asset is a 3D mesh
