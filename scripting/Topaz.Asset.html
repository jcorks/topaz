 Pure assets dont exist, but serve 
 symbolically as a basis for these

Topaz.Asset.TYPE 

 Type of asset.

 Every asset has a type that gives a hint to what child class
 it is.
NONE The asset doesnt have a type. This is used for error cases.
IMAGE The asset is a Topaz.Image, which holds raw pixel data.
SOUND The asset is a Topaz.Sound, which holds raw audio data.
MATERIAL The asset is a Topaz.Material, which defines how 3d geometry looks.
DATA The asset is a Topaz.Data, which expresses raw bytes / text.
MESH The asset is a Topaz.Mesh, which holds raw 3D data.
